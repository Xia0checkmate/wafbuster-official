Engine

7-Phase Pipeline

Each phase feeds intelligence into the next. Nothing runs blind.

Phase 01
Connectivity + WAF Detection
10 attack payloads, 17 vendor-specific response headers, Server signature map for 10 WAF vendors, and a Rate Limit Test firing 12 rapid requests to catch throttle-based WAFs.
10 payloads17 headersrate limit test
Phase 02
Smart Favicon Discovery
5-strategy engine: standard <link rel="icon">, fuzzy keyword scan for any href containing "favicon", Web App Manifest parsing, /favicon.svg probe, then /favicon.ico fallback.
5 strategieskeyword scanMurmurHash3
Phase 03
Subdomain Enumeration
4 parallel passive sources: subfinder, assetfinder, crt.sh certificate transparency, HackerTarget. Smart relevance filter removes wildcards and off-target domains before IP resolution.
4 sourcessmart filter
Phase 04
DNS History + IP Collection
Concurrent subdomain-to-IP resolution. Pulls historical DNS records from HackerTarget to surface IPs that were exposed before the CDN was deployed.
DNS historyconcurrent
Phase 04.5
ASN + IP Prefix Filtering
Two-layer filter: instant IP prefix table (60+ Cloudflare, Akamai, Fastly, DDoS-Guard ranges — no API call), then live ASN lookup across 60+ WAF/CDN autonomous systems.
60+ ASNsIP prefix tablezero noise
Phase 05
Multi-Strategy Origin Probe
12 uTLS browser profiles (Chrome, Firefox, Safari, Edge, Randomized) × SNI toggle × TLS 1.0→1.3 cycling × HTTP Host injection. SSL certificate CN + SAN matching. False-positive filter built in.

Phase 05.5
Second-Order Bypass
Fires on 401/403/418. 17 spoof headers × 6 IP groups (loopback, private, CDN-trusted, link-local 169.254.x, reflective) × 25 paths including URL-encoded variants and sensitive endpoints.
